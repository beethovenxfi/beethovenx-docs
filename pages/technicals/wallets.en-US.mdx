## Ecosystem Wallets
The following wallets are used to manage the corresponding funds:
- [Team Fund Multisig Wallet](https://ftmscan.com/address/0x0edfcc1b8d082cd46d13db694b849d7d8151c6d5#code)

The Balancer admin has the ability to change protocol fees or sweep the protocol fees:
- [Balancer admin FTM](https://ftmscan.com/address/0x9d0327954009C59eD70Dc98b7726e911879d4D92)
- [Balancer admin OP](https://optimistic.etherscan.io/address/0xdCB017f63aCd34DD032A702315459ceCc05536DD)

- [Balancer operator FTM](https://ftmscan.com/address/0xCd983793ADb846dcE4830c22F30C7Ef0C864a776)
- [Balancer operator OP](https://optimistic.etherscan.io/address/0xd9e2889AC8C6fFF8e94c7c1bEEAde1352dF1A513)

- [Gauge operator OP](https://optimistic.etherscan.io/address/0xf56D2a5faA313f597812dD0EDF7f27F386Aa0913)
